## 🛠️ Troubleshooting & Common Issues

### The app won't install on my iPhone

Ensure you are using Safari (not Chrome or Firefox) to download the configuration profile. Navigate to **Settings → General → VPN & Device Management** and manually trigger the installation from there. Restart your device if the profile does not appear.

### "Untrusted Developer" warning on iOS

This is normal for sideloaded applications. Go to **Settings → General → VPN & Device Management**, find the developer profile, and tap **Trust**. You will need to confirm the action.

### The app crashes on launch (Android)

Clear the app cache through **Settings → Apps → [App Name] → Storage → Clear Cache**. If the problem persists, uninstall and reinstall the latest version. Ensure your Android version is 8.0 or higher.

### Features are not loading or responding

Check your internet connection. Some features require an initial server handshake on first use. If you are behind a restrictive firewall or network filter, try switching to mobile data temporarily.

### Cloud sync is not working

Verify that you are signed in with the same account on all devices. Check that **cloud_sync** is set to **enabled** in your configuration file. Restart the app after making changes.

### High battery drain

Reduce the `execution_threads` parameter in your configuration to 2 or lower. Disable background refresh in your device's battery settings for the app. Close the app fully when not in use.

### The built-in library is empty

This usually means the initial content download was interrupted. Connect to a stable Wi-Fi network, open the library tab, and pull down to trigger a manual refresh.
